### Installation

  - Using [pip](https://en.wikipedia.org/wiki/Pip_(package_manager)):

```bash
pip install imagedominantcolor -U
```

  - Install directly from GitHub:

```bash
pip install git+https://github.com/akamhy/imagedominantcolor.git
```
